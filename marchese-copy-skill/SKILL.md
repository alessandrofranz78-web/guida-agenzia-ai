# Marchese Hypnotic Copy — Skill Operativa

Framework di copywriting ipnotico estratto dai materiali di Marcello Marchese.
Ogni tecnica include: nome, spiegazione, quando usarla, e come applicarla.

## WORKFLOW DI APPLICAZIONE

Quando scrivi o revisioni copy, segui questo ordine:

1. **HOOK** (primi 3 secondi) → usa Power Hook o Loop Aperto
2. **AGITAZIONE** → usa Frame Bianco-Nero o Noi Contro Loro
3. **CREDIBILITÀ** → usa Eroe Riluttante + Frame Precisione
4. **DESIDERIO** → usa Scappatoia del Segreto + Fascinations (Regola del Tre)
5. **AZIONE** → usa Bind Complesso o Aiuto Disinteressato per il CTA
6. **VIRALITÀ** → usa Frame Grandi Numeri + Social Payment Nobile

Non tutte le tecniche servono in ogni pezzo. Scegli quelle appropriate al formato:

| Formato | Tecniche prioritarie |
|---------|---------------------|
| Headline/Hook | Power Hook, Loop Aperto, Regola del Tre |
| Landing page | Tutte in sequenza |
| Post social | Power Hook + 1 leva emotiva + CTA |
| Email | Loop Aperto + Eroe Riluttante + CTA |
| CTA button | Bind Complesso o Frame Bianco-Nero |
| Share text | Loop Aperto + Frame Precisione + Fascinations |
| Gate/Paywall | Eroe Riluttante + Aiuto Disinteressato |
| Video script | Power Hook + Noi Contro Loro + Loop Aperto |

## 1. PATTERN IPNOTICI (Cattura attenzione)

### 1.1 Power Hook (Hook nell'hook)
**Cosa:** Frase brevissima e scioccante con Power Words. Cattura in <3 secondi.
**Quando:** Apertura di QUALSIASI testo (post, email, landing, video).
**Come:** Max 4-5 parole. Emozione forte. Punto. Poi spazio.
**Power Words:** derubato, furioso, terrorizzato, ossessionato, devastante, segreto, proibito, gratuito, istantaneo, garantito, esclusivo, limitato, scioccante.

```
❌ "Oggi vi parlo di un argomento interessante"
✅ "Mi sentivo derubato."
✅ "Ero furioso."
✅ "Cattive notizie."
✅ "Ho fatto un errore da €1.702."
```

### 1.2 Loop Aperto (Power Story Incompleta)
**Cosa:** Apri una storia o domanda, interrompila PRIMA della conclusione.
**Quando:** Per costringere il click, la lettura, o la condivisione.
**Come:** Dai abbastanza contesto per creare curiosità, poi taglia. Il cervello DEVE chiudere il loop.

```
❌ "In questa guida trovi 11 capitoli su come usare l'AI"
✅ "Ho speso €1.702 testando ogni AI sul mercato. Ma il segreto non è la cifra. È la riga nel capitolo 3 che..."
✅ "Stavo per chiudere tutto. Poi alle 3 di notte ho aperto n8n e..."
```

### 1.3 Bind Complesso (Presupposizione)
**Cosa:** Due domande in sequenza rapida. La prima presuppone un fatto spiazzante, la seconda chiede l'azione.
**Quando:** CTA, bottoni, chiusure di post.
**Come:** La prima domanda distrae il "guardiano critico". Il lettore risponde solo alla seconda.

```
❌ "Vuoi scaricare la guida?"
✅ "Sapevi che i tuoi concorrenti usano già l'AI per rubarti clienti? Condividi questo post per scoprire come fare meglio di loro."
✅ "Sai quante ore sprechi ogni settimana in task che un agente AI farebbe in 3 minuti? Scarica la guida e conta."
```

## 2. LEVE EMOTIVE (Spinta all'azione)

### 2.1 Frame Bianco-Nero (Scelta Illusoria)
**Cosa:** Due sole alternative estreme. Una terribile, una perfetta.
**Quando:** CTA, decisioni binarie, chiusure urgenti.
**Come:** Nessuna via di mezzo. O agisci o soffri.

```
❌ "Potresti trovare utile questa guida"
✅ "O accedi ora a 31 notti di test condensate in 11 capitoli, o domani sarai ancora a fare tutto da solo mentre i tuoi competitor automatizzano."
```

### 2.2 Noi Contro Loro (Nemico Comune)
**Cosa:** Allearsi col lettore contro un nemico comune (guru, agenzie, multinazionali).
**Quando:** Positioning, differenziazione, creazione di community.
**Come:** Punta il dito contro chi tiene nascoste le soluzioni o ne approfitta.

```
❌ "L'AI è accessibile a tutti"
✅ "Le web agency ti fanno credere che l'AI sia complessa per giustificare le loro parcelle. In 11 capitoli gratuiti, distruggiamo il loro giocattolo."
```

### 2.3 Scappatoia del Segreto d'Élite (FOMO Elegante)
**Cosa:** L'informazione è in mano a pochi. Chi non agisce resta fuori.
**Quando:** Scarsità, esclusività, community chiuse.
**Come:** Non dire "sbrigati". Dì "questa cerchia si sta chiudendo".

```
❌ "Scarica la guida prima che finisca"
✅ "Questi 11 capitoli sono già nelle mani di centinaia di imprenditori. Il gruppo si sta chiudendo."
```

## 3. STRUTTURE DI COPY (Costruzione del messaggio)

### 3.1 Frame Precisione (Credibilità Estrema)
**Cosa:** Numeri specifici con decimali. Mai numeri tondi.
**Quando:** SEMPRE quando citi dati, risultati, prezzi, tempi.
**Come:** €1.702 > €1.700. 31 notti > "un mese". 8 progetti > "diversi progetti".

```
❌ "Ho testato tanti strumenti AI per un mese"
✅ "31 notti. €1.702,45 di abbonamenti. 14 piattaforme AI testate una per una."
```

### 3.2 Regola del Tre (Fascinations)
**Cosa:** Tre elementi/promesse in sequenza. Crea un "suono" definitivo nel cervello.
**Quando:** Bullet point, headline, liste di benefici.
**Come:** Tre item, ritmo crescente, l'ultimo è il più forte.

```
❌ "Con questa guida impari tante cose utili sull'AI"
✅ "Azzerare i costi. Automatizzare il lavoro. Distruggere la concorrenza."
✅ "11 capitoli. 12 prompt pronti. 30 giorni per cambiare tutto."
```

### 3.3 L'Eroe Riluttante (Prova Anti-Venditore)
**Cosa:** Ammetti di NON essere un guru. Sei una persona come il lettore che ha trovato qualcosa.
**Quando:** Bio, presentazioni, aperture di sales page, email di benvenuto.
**Come:** Abbassa le difese. "Non sono X. Sono Y come te. E ho scoperto Z."

```
❌ "Sono un esperto di AI con anni di esperienza"
✅ "Non sono un formatore AI. Sono un imprenditore che ha sudato €1.702 e 31 notti per capirci qualcosa. E ora te lo regalo."
```

## 4. TECNICHE DI VIRALITÀ (Far condividere)

### 4.1 Frame Grandi Numeri (Riprova Sociale Primitiva)
**Cosa:** Mostra che una massa di persone simili al target sta già agendo.
**Quando:** Social proof, CTA di condivisione, landing page.
**Come:** "Centinaia di...", "Il 73% dei...", "Ogni giorno X persone..."

```
❌ "Scarica la guida"
✅ "Centinaia di imprenditori stanno già usando questi 11 capitoli. Unisciti a loro."
```

### 4.2 Aiuto Disinteressato (Social Payment Nobile)
**Cosa:** Trasforma la condivisione in una missione nobile. Non condividi per te — lo fai per gli altri.
**Quando:** Gate di condivisione, CTA virali, referral.
**Come:** "Qualcuno nella tua rete ne ha bisogno. Faglielo sapere."

```
❌ "Condividi per sbloccare"
✅ "Ti chiedo solo un post. Qualcuno nella tua rete sta lavorando 60 ore a settimana senza sapere che esiste un'alternativa. Faglielo sapere."
```

## 5. CHECKLIST DI REVISIONE COPY

Quando revisioni un testo, verifica questi punti nell'ordine:

```
□ HOOK: Le prime 5 parole colpiscono allo stomaco?
□ LOOP: C'è almeno un loop aperto che costringe a continuare?
□ NUMERI: Sono specifici (€1.702) o generici (€1.700)?
□ NEMICO: C'è un "loro" contro cui ci alleiamo col lettore?
□ EROE: L'autore si presenta come pari, non come guru?
□ TRE: I benefici sono raggruppati a tre?
□ BIANCO-NERO: Il CTA offre solo due scelte (agire o perdere)?
□ NOBILE: La condivisione è presentata come atto altruista?
□ FRASI: Sono corte? Max 10-12 parole per frase?
□ TECNO-ARAMAICO: Zero gergo tecnico non tradotto?
```

Se almeno 3 punti mancano → il copy va riscritto.
Se mancano 1-2 → interventi chirurgici sui punti deboli.
Se tutti presenti → il copy è pronto per il test.

## 6. TEMPLATE RAPIDI

### Post Social (formato corto)
```
[POWER HOOK — 3-5 parole shock]

[STORIA BREVE — 2-3 frasi con numeri precisi]

[FASCINATIONS — 3 benefici in Regola del Tre]

[CTA — Bind Complesso o Frame Bianco-Nero]

[HASHTAG]
```

### Email Subject Line
```
[POWER HOOK] + [LOOP APERTO]
Esempio: "Mi sentivo derubato. Poi ho aperto il capitolo 3..."
```

### CTA Button
```
[AZIONE] + [BENEFICIO IMMEDIATO]
Esempio: "ACCEDI AI 31 GIORNI DI TEST →"
Non: "Scarica" / "Invia" / "Submit"
```

### Share Text (per condivisioni social)
```
[FRAME PRECISIONE — numeri specifici] + [LOOP APERTO — curiosity gap] + [REGOLA DEL TRE — benefici]
Esempio: "Un imprenditore ha passato 31 notti insonni e speso €1.702 testando ogni AI. Il capitolo 3 da solo vale il prezzo di un corso. Zero codice, zero fuffa — solo quello che funziona."
```

## REGOLE D'ORO (NON NEGOZIABILI)

1. **Mai "noi" aziendale.** Sempre "io" e "tu".
2. **Mai frasi oltre 12 parole** nel copy persuasivo.
3. **Mai numeri tondi.** Sempre specifici.
4. **Mai vendere direttamente.** Sempre educare + dimostrare valore prima.
5. **Mai gergo tecnico** senza traduzione in beneficio concreto.
6. **Mai CTA passivi** ("scarica", "clicca qui"). Sempre azione + risultato.
7. **Mai aprire con una spiegazione.** Sempre con una storia o un colpo.
